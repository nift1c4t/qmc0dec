# qmc0dec

This is a qmc0/qmc3/qmcogg/qmcflac file decoder ported from https://github.com/Presburger/qmc-decoder.

## Install 

```bash
git clone https://github.com/nift1c4t/qmc0dec.git
cd qmc0dec
pip install .
```
